🐕 Dog Running Game

A simple Endless Runner game where a dog keeps running forward, avoids obstacles, and collects bones to increase score.
This game is built using HTML, CSS, and JavaScript.

✨ Features

🐶 Dog character that continuously runs forward

⚡ One-tap jump controls (Spacebar / Click / Touch)

🚧 Different obstacles to avoid

🦴 Collect bones to increase score

📈 Speed increases as the score goes up

🎨 Clean UI with score, points, and game over screen
